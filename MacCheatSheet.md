# My personal Mac cheat sheet

## System

### Applications

* Eclipse
* iTerm as shell

### Command aliases

Add system aliases: `nano ./zshrc`

Like this:
```shell
alias ll="ls -la"
alias pullall="ls | xargs -I{} git -C {} pull"
```

## Character Bindings

* Tilde char (`~`) - Alt/Option + n, followed by Space
* Backslash (``) - Alt/Option + Shift + 7
